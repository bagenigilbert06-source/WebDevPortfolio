# WebDevPortfolio

A responsive React portfolio for full-stack developer Bageni Gilbert.

## Development

This repository uses pnpm workspaces. Run commands from the repository root:

```bash
pnpm install
pnpm dev
```

The development server is available at `http://localhost:3000`.

## Commands

- `pnpm dev` starts the development server.
- `pnpm build` creates an optimized production build.
- `pnpm test` runs the test suite.

Application source code lives in `profolio/src`.
